Polars, the fastest DataFrame library you never heard of...

This talk will introduce Polars, a blazingly fast DataFrame library written in Rust on top of Apache Arrow. It's a DataFrame library that brings exploritory data analysis closer to the lessons learned in database research. And because ot that it is the library that puts python in top for OLAP queries. See the excellent query benchmark performances.
