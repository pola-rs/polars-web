Knowing of data wrangling habits, Polars exposes a complete Python API, including the full set of features to manipulate DataFrames in eager and/or lazy ways, using an expression language that will empower you to create readable performant code.

DataFrames to the Rust ecosystem

Polars is written from scratch in Rust, not compromosing in its decision to provide a feature complete DataFrame API to the Rust ecosystem. Use it as a DataFrame library or as query engine backend for your data model.

On the shoulders of a giant

Apache Arrow is used for all the in-memory data management, enabling Polars to improve resource use and processing performance. By using Arrow polars integrates seamlessly in with other tools in the arrow landscape.

Welcome to fast data wrangling

Polars is a lightning fast DataFrame library/ in-memory query engine. Its embarrassingly parallel execution, cache efficient algorithms and expressive API make it perfect for data wrangling, ETL pipelines snappy API's and much more.

Quickstart

Rust

Below a quick demonstration of loading the Iris dataset as a Rust DataFrame. 


use polars::prelude::*;

fn example() -> Result<DataFrame> {
    let df = df![
        "A"=> [1, 2, 3, 4, 5],
       "fruits"=> ["banana", "banana", "apple", "apple", "banana"],
        "B"=> [5, 4, 3, 2, 1],
        "cars"=> ["beetle", "audi", "beetle", "beetle", "beetle"]
    ]?;

    df.lazy()
        .sort("fruits", false)
        .select([
            col("fruits"),
            col("B").filter(col("cars").eq(lit("beetle"))).sum(),
            col("A")
                .filter(col("B").gt(lit(2)))
                .over(vec![col("cars")])
                .alias("sum_A_by_cars"),
            col("A")
                .sum()
                .over(vec![col("cars")])
                .alias("sum_A_by_fruits"),
            col("A")
                .sort_by(col("B"), false)
                .over(vec![col("fruits")])
                .flatten()
                .alias("rev_A_by_fruits"),
        ])
        .collect()
}

Python

Below a quick demonstration of reading, filtering, grouping and aggregating the famous Iris dataset. 


import polars as pl
df = pl.DataFrame(
    {
        "A": [1, 2, 3, 4, 5],
        "fruits": ["banana", "banana", "apple", "apple", "banana"],
        "B": [5, 4, 3, 2, 1],
        "cars": ["beetle", "audi", "beetle", "beetle", "beetle"],
    }
)

(df
    .sort("fruits")
    .select([
      "fruits",
      col("B").filter(col("cars") == "beetle").sum(),
      col("A").filter(col("B") > 2).sum().over("cars").alias("sum_A_by_cars"),       # groups by "cars"
      col("A").sum().over("fruits").alias("sum_A_by_fruits"),                        # groups by "fruits"
      col("A").sort_by("B").over("fruits").flatten().alias("sort_A_by_B_by_fruits")  # groups by "fruits"
])
