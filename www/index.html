Welcome to fast data wrangling

Polars aims at being a lightning fast DataFrame library utilizing all available cores on your machine. Its ideal use case lies in data too big for Pandas (for instance) but too small for Spark. If you however need to process data that does not fit in the memory of a single machine (even after filtering), Polars is not the solution to your problem.

Polars consists of an eager API that is similar to Pandas: any operation is immediately executed and produces a result. In addition, and similarly to Spark, Polars comes with a query planner (also referred as lazy API) that may -and probably will- optimize and parallelize your data processing, decreasing the time spent running the workload and reducing memory usage.

Quickstart

Rust

Below a quick demonstration of loading the Iris dataset as a Rust DataFrame.

use polars::prelude::*;

fn example() -> Result {
    CsvReader::from_path("iris.csv")?
            .infer_schema(None)
            .has_header(true)
            .finish()
}

Python

Below a quick demonstration of reading, filtering, grouping and aggregating the famous Iris dataset.

import polars as pl

q = (
    pl.scan_csv("iris.csv")
    .filter(pl.col("sepal_length") > 5)
    .groupby("species")
    .agg(pl.col("*").sum())
)

df = q.collect()
